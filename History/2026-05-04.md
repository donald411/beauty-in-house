## 5. Hero 섹션의 여자 이미지 제거

대문(Hero) 배경에 사용되던 `images/Makeup1.gif` (메이크업 받는 여자 이미지)를 제거.

수정 위치 (`css/style.css`):
- `.hero-bg` 의 `background-image: url('../images/Makeup1.gif')` 및 관련 `background-size / position / repeat / opacity / transform / animation` 속성 제거 → `background-color: var(--primary)` 단색으로 단순화
- `heroZoom` 애니메이션은 더 이상 호출되지 않음 (정의는 남아있으나 무해)
- 모바일 반응형의 `.hero-bg { background-size: 140%; ... }` 라인 제거

기존의 `.hero-overlay` 그라디언트(검정 → 핑크)는 그대로 유지되어 Hero 영역의 톤은 보존됨.
